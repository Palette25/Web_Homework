一些游戏特性：
1.终端输入node server.js即可开始游戏，访问localhost:3000/Sn/index.html， 其中n为1到5的数字
2.S1只支持一次hover入@标记之后，只能计算一次，移出后计算器重置
3.S5的一些特性：
  （1）大气泡上方显示对应的message，具体的对应message请自行查看wiki
   “才怪”的反面是“是的是的”。
  （2）游戏中handler会随机出现错误，出现之后也会向服务端请求数据，并继续执行相应的逻辑，
  并且一旦出现异常错误之时，作为游戏特色，message的右方将会出现截止到当前为止的计算结果之和。
  （3）S5的大气泡显示随机序列，最终的总和作为最后一条message的结果数值输出
